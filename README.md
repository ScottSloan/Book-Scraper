# 小说爬取工具 - Book Scraper
使用 Python 编写的一款轻量级的小说爬取工具  
Latest Version 1.31  
更新日期：2021-12-11

## 功能
* 支持`GUI`图形界面 (wxPython)  
* 支持`搜索小说`功能  
* 支持`收藏小说`功能 
* 支持`多线程`爬取小说  
* 支持`代理IP`池 
* 支持`阅读模式`

## Version 1.31 新增内容 (2021-12-11)
* 支持`阅读模式`，可将小说保存到本地阅读
* 支持`首选项`功能，可修改代理 IP 和线程池有关设置
* 修复一些已知问题
